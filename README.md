# Humanos Digitais — Website

Website oficial da plataforma Humanos Digitais com Sofia AI como avatar host.

## Características

- **Sofia AI**: Avatar inteligente com sync-lip (HeyGen/D-ID)
- **14 Avatares**: Personagens por segmento (corporativo, entretenimento)
- **RAG Integrado**: Base de conhecimento com 20 documentos
- **Conversas Interativas**: Chat com limite de 5 interações (demo)
- **Leads & WhatsApp**: Integração com CRM

## Stack

- **Frontend**: Next.js 14, Tailwind CSS, Framer Motion
- **Backend**: Supabase (PostgreSQL + Storage)
- **RAG**: Embeddings + pgvector/Pinecone
- **Avatares**: HeyGen API / D-ID API

## Instalação

```bash
npm install
npm run dev
```
